## Fable / Claude originals

- **Interview capacity as a triage gate** — *2026-07-16, workshop playtest-4 barrage (Q A3).*
  Interviewing candidates is Attention-gated, so a large applicant pool cannot all be
  screened — the player must **triage who is worth an interview**. This turns
  "advertise for volume" into a real gamble (more applicants than you can afford to see)
  and makes the hiring pool a scarcity problem, not just a list. Pip: *"sounds good to me,
  great contribution!"* Feeds the L2 hiring pipeline (candidate-card info model,
  hire-as-scouting, ADR-0004).
