# P(Doom)1 -- v0.11.0 (first friends & family build)

> DRAFT release notes for Pip's voice-pass. This becomes the GitHub Release body.
> Framing: friends & family direct download; Windows only; Steam + web port later.

You run an AI safety lab. It's a turn-based strategy game about doing the paperwork
that might save the world -- manage doom, reputation, money, papers, governance, and
compute; build a team; and try to keep humanity in the game a little longer.

## What's in this build
- **The full core loop.** Plan in turns, run the months, watch doom and burnout climb,
  and make the calls that decide how long the lab lasts.
- **Hiring pipeline.** Source, interview, make offers, and onboard candidates -- each with
  hidden appetites and quirks you only learn by working with them.
- **Two views.** PLAN and WATCH, toggle any time (key V) -- decide, then watch it play out.
- **Honest clocks and honest endings.** A clean turn-and-date readout, and defeat screens
  that name the thing that actually got you (credibility collapse, doom, bankruptcy),
  not a canned line.
- **A feed you can live with.** The event stream has an arxiv-flood filter so real,
  actionable events aren't buried.
- **The leaderboard loop.** Your score propagates to the world board -- play a run and
  watch your name (and your friends') show up.
- **Deterministic runs.** Seeded, so a given seed plays the same and scores compare fairly.

## Known limits (it's an alpha, and we'd rather be honest)
- **Music is placeholder.** The adaptive doom-band system is wired in; the real score is
  a few studio hours away.
- **Art is mid-iteration.** Office props and characters are actively being refined.
- **Windows only for now.** Steam when the game's more stable; a web port in response to
  what you all tell us.

## How to play
Download, unzip, run **PDoom.exe**. That's it.

## Found a bug?
Use the in-game bug reporter (it files a GitHub issue), or open one directly. This is a
friends & family build -- your rough edges are exactly what we want to hear about.
